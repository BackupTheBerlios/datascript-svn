Roadmap

  • The parser must be more robust on syntax errors and always log error messages instead of throwing exceptions.
  • Single imports: import com.acme.foo.MySequence.
  • Subtype constraints.
  • Outline and Go to declaration action for the Eclipse editor plugin.
  • There are plans to implement an alternative C++ backend using a data-driven approach instead of generating a C++ class per DataScript type.
