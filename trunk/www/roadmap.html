Roadmap

  • The relational extensions need to be documented.
  • We require a plugin interface for the C++ backend.
  • Parsing of doc comments needs to be cleaned up.
  • The parser must be more robust on syntax errors and always log error messages instead of throwing exceptions.
  • We need limited support for functions in order to assign integer values to compound types, e.g. for variable length integers.
  • Single imports: import com.acme.foo.MySequence.
  • Subtype constraints.
